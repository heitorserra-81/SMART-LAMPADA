Rotinas

O app não fica rodando sozinho em segundo plano — quem dispara por horário ou chegada é o app Atalhos do iPhone, na aba "Automação". Aqui você pega o link certo de cada cena pra colar lá.
Como criar uma rotina automática:
  1. Copia o link de uma cena abaixo (ou de Início/Ajustes)
  2. Abre o app Atalhos → aba Automação
  3. Toca em "+" → "Criar Automação Pessoal"
  4. Escolhe o gatilho: Hora do Dia, Chegada, etc.
  5. Ação: "Obter Conteúdo de URL" → cola o link
  6. Desativa "Perguntar Antes de Executar"
Cenas disponíveis pra rotina
Crie cenas na aba "Cenas" primeiro — elas aparecem aqui prontas pra virar rotina.

Ajustes

Links rápidos para Atalhos

Como criar um Atalho com um link

  1. Copia o link desejado acima (toca em "Copiar")
  2. Abre o app Atalhos → toca em "+"
  3. "Adicionar Ação" → busca "Obter Conteúdo de URL"
  4. Toca no campo da URL → Colar
  5. Nos "..." no topo → dá um nome e ativa "Usar com Siri"
  6. Grava a frase que você quer falar

Um único comando de voz pra várias cores

  1. Cria um Atalho novo, nome "Mudar cor da luz"
  2. Adiciona ação "Perguntar Entrada" (texto)
  3. Adiciona blocos "Se": Entrada = "azul" → link azul; Entrada = "vermelho" → link vermelho; repete pra cada cor
  4. Ativa "Usar com Siri", frase: "mudar cor da luz"
  5. Ao usar: "Ei Siri, mudar cor da luz" → ela pergunta → você fala a cor
